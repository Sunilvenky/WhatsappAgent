## 🐛 Troubleshooting

### Pages Not Loading?
1. Check browser console for errors
2. Verify `localhost:5173` in address bar
3. Ensure `npm run dev` is running
4. Check network tab for failed requests

### API Calls Failing?
1. Verify backend is running on `localhost:8000`
2. Check `VITE_API_URL` environment variable
3. Inspect browser console for error details
4. Check network tab for response status

### Styling Issues?
1. Run `npm install` to ensure Tailwind
2. Clear browser cache
3. Restart dev server (`npm run dev`)
4. Check `tailwind.config.js` exists

### State Issues?
1. Verify providers wrap routes in `App.jsx`
2. Check localStorage for tokens
3. Inspect React DevTools component tree
4. Check context values in console
